Why KernChan?

No Permanent Changes

Loaded entirely into RAM. A single reboot wipes the module completely, leaving your system pristine.

Command Dispatch

Built-in parser handles info, mem, uptime, help – easily extendable.

Safe by Design

copy_from_user, mutex locks, and strict permissions (0660) keep your kernel secure.

One‑Click Install

Download install.sh, push to device, and execute. Pre‑built for your exact kernel config.

Supported Commands
Command Description
info Show kernel release, version, and architecture
mem Display total, free, and available RAM (MB)
uptime System uptime in hours / minutes / seconds
help Print this command reference

Extend it easily: add your own handler in cmd_table[].

How to Use
1
Download the installer
Grab install.sh from the Actions tab of the repository.
2
Push to your device
adb push install.sh /data/local/tmp/
3
Load the module (root required)
su -c "sh /data/local/tmp/install.sh"
4
Start talking to the kernel
echo "info" > /dev/kern_chan && cat /dev/kern_chan
