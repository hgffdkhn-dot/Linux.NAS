kernchan a temporary kernel channel for Android

  • loads into RAM, gone after reboot
  • works on Redmi K40 Gaming (4.14.186)
  • speaks via /dev/kern_chan
  • pre‑built, one‑script install

You send a command, it answers with kernel info. No permanent changes, no risky hacks – just a clean pipe to the inside. 

$ echo "info" > /dev/kern_chan
$ cat /dev/kern_chan
Kernel: Linux 4.14.186 #1 SMP ...

✦ You can add your own commands by extending the cmd_table[] in the source.

how to get it running

  1. Grab the installer – download install.sh from the Actions tab (latest build).
  2. Push to your phone – adb push install.sh /data/local/tmp/
  3. Load it (root) – su -c "sh /data/local/tmp/install.sh"
  4. Talk to kernel – echo "info" > /dev/kern_chan && cat /dev/kern_chan

That’s it. The module lives only until you reboot – after that, it’s gone.

notes & warnings

  • ⛔ Root required – insmod needs superuser.
  • 🔒 Use at your own risk – kernel access is powerful; keep it to trusted scripts.
  • 🧹 Clean unload – a reboot is the simplest way to remove everything.
  • 📦 Built for your kernel – the pre‑built module matches the 4.14.186 config from this device.
